## 策略说明

### 智能马丁格尔策略

- **核心逻辑**: 基于亏损加仓的资金管理策略
- **风险控制**: 设置止损线和最大仓位上限
- **适用场景**: 震荡行情、反弹预期强的标的

### 智能短期趋势策略

- **核心逻辑**: 结合技术指标判断短期走势
- **买入信号**: 多头排列、成交量放大、突破关键位
- **卖出信号**: 空头排列、跌破支撑、背离现象


## 使用示例

### 1. 查询特定用户的账户信息

```bash
python scripts/stock_redis_query.py --username <用户名> --type account
```

### 2. 查询策略执行记录

```bash
python scripts/stock_redis_query.py --username <用户名> --type strategy
```

### 3. 下单前审核

提供订单参数，返回审核意见：
- 价格合理性
- 仓位限制检查
- 止损止盈建议


## 注意事项

1. **Redis 依赖**: 需要配置正确的 Redis 连接信息
2. **数据来源**: 以 vnpy 实际运行数据为准，Redis 仅作辅助
3. **风险提示**: 所有分析结果仅供参考，不构成投资建议
4. **权限控制**: 确保仅授权的账号可访问敏感数据
